Introduction
1. Each annotator has an unique user id and password, please take care of it.
2. Every task will include at least one question-and-answer pair(QA pair). You need to read the whole context and image on the left and then assess each QA pair on the right.
3. Correct the grammatical errors of each question and write the whole corrected question in the inputbox, otherwise, leave it with None (Wrong Capitalization is not counted as an error).
4. Assess each answer with the checkboxes according to the context and image.

4.1 Check the checkbox if the answer is incorrect to the question.

4.2 Check the checkbox if the answer is not unique to the question. The answer is not unique if it satisfies all following conditions:

1) You can find other answers from the original context.

2) The other answers and the one we provided are not referring to the same object. e.g. Donald Trump is not counted as other answers to Trump because they are referring to the same person.

4.3 Check the checkbox if you can determine the answer without looking at the context.

4.4 Check the checkbox if the answer appears in the image.

4.5 Check the checkbox if you can determine the answer without looking at the image.

5. Click the refresh button to clear all your selections.
6. If the annotations are not submitted successfully by clicking the submit button, please if your answers are still there.
